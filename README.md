<h3 align="center">EEG Signal Classification for Sleep Disorder Detection</h3>

###

<h4 align="left">Developed an advanced deep learning system for classifying EEG signals to assistin sleep disorder diagnosis. The project involved processing raw neurological data, handling severe class imbalance, and implementing a hybrid CNN-LSTM architecture<br>that achieved state-of-the-art performance in sleep stage classification.<br><br>• Engineered a comprehensive data pipeline including signal normalization (StandardScaler), class balancing (SMOTE), and specialized preprocessing fortime-series EEG data. Implemented custom evaluation metrics, including a weighted F1-score callback for real-time performance monitoring during model training.<br><br>• Designed and trained a novel neural network architecture combining 2D convolutional layers for spatial feature extraction with LSTM layers for temporal pattern recognition. The model incorporates advanced techniques, including batch normalization, strategic dropout, and dynamic learning rate adjustment for optimal performance.<br><br>• My primary contributions included designing the data preprocessing pipeline, implementing the SMOTE oversampling strategy for imbalanced classes, developing the custom F1-score evaluation callback, and optimizing the hybrid CNN-LSTM architecture</h4>
